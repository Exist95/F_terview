## 커밋 컨벤션
### 예시) [STYLE]: 스타일 변경

- [FEAT]: 새로운 기능 (기능 구현)
- [FIX]: 버그 수정
- [DOCS]: 문서 변경 (README 등등)
- [STYLE]: 스타일 변경
- [REFECTOR]: 리팩토링에 대한 커밋
- [TEST]: 누락된 테스트 추가 또는 기존 테스트 수정
- [CHORE]: 코드 수정 없이 설정을 변경, 패키지 설치
